# Comments

![Launcher icon](app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

Demo project. Not maintained.


## How to run

``` bash
$ ./gradlew clean installDebug
```

## Architecture in modules

- api - http layer
- app - user interface
- net - scheduling jobs
- store - persistence


## Libraries used

- androidJob
- assertjCore
- constraintLayout
- junit
- kotlin
- kotlinxCoroutines
- okhttp
- moshi
- retrofit
- room
- rxandroid
- supportLibrary
